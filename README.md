# MotoGrep2
Xposed Module grepein SystemUI MotoG 5.0.2

versi 1.0 beta 1

# Fitur
* geser-geser layout statusbar
* ganti icon statusbar dan navbar dari png di sdcard
* ganti icon notif selain SystemUI (ex: sms, bbm, etc)
  format nama png: package.name#res_drawable_name.png
  (ex sms: com.android.mms#stat_notify_sms.png)

# Dependencies
* xposedbridge 60+ (5.0)
* android support v4
* android support v7 - AppCompat
* android support v7 - RecyclerView
